# JARVIS • AI软件研发 
    专家级AI软件研发助手，JARVIS引领编程未来！

### JARVIS • AI软件研发是什么？

JARVIS • AI软件研发是一个专家级AI软件研发助手，主要目标是辅助开发人员更快速、更低门槛地设计和编写代码。目前主要针对Java代码的生成。

核心功能包括：
1. 三种低代码生成方式：
    - 数据库设计生成
    - 接口设计生成
    - AI问答代码生成
2. 智能代码补全 - 自动提示代码
3. Agent模式开发 - 通过工具面板与AI聊天，对代码进行生成、修改与调试
4. 多模型支持 - 支持Claude、Qwen、GLM、Kimi、DeepSeek等主流模型
5. 数据库功能：
    - MySQL和MongoDB支持
    - SQL执行和结果展示
    - 数据库设计和管理
